# Contributing to 90storezon Calculator Website

Thank you for your interest in contributing to 90storezon! We welcome contributions from everyone.

## Code of Conduct
By participating in this project, you agree to maintain a respectful and inclusive environment for all contributors.

## How to Contribute

### 1. Report Bugs
If you find a bug, please create an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable
- Browser/OS information

### 2. Suggest Features
We welcome feature suggestions! Please include:
- Use case for the feature
- Expected behavior
- Any mockups or examples

### 3. Submit Pull Requests

#### Development Setup
1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/90storezon.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test thoroughly
6. Commit: `git commit -m "Add: description of changes"`
7. Push: `git push origin feature/your-feature-name`
8. Create a Pull Request

#### Coding Standards
- Follow PSR-12 for PHP code
- Use meaningful variable and function names
- Add comments for complex logic
- Write unit tests for new features
- Keep functions focused and small
- Use prepared statements for database queries
