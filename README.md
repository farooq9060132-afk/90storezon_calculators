# 90storezon Online Calculators

A comprehensive collection of 50+ free online calculators for finance, health, math, and more.

## Project Overview

This repository contains a PHP-based web application featuring over 50 online calculators organized into different categories. The calculators are designed to be responsive, user-friendly, and completely free to use without registration.

## Features

- **50+ Calculators**: Covering financial, health, math, and utility categories
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Registration Required**: All calculators are free to use
- **Clean URLs**: User-friendly URLs without numeric prefixes
- **Modern UI**: Consistent design across all calculators

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/90storezon-calculators.git
   ```

2. Upload files to your web server with PHP support

3. Ensure URL rewriting is enabled (Apache with mod_rewrite)

4. The calculators should now be accessible through your web server

## URL Structure

The calculators use clean URLs without numeric prefixes:
- `/calculators/loan-calculator/` (not `/calculators/01-loan-calculator/`)
- `/calculators/anorexic-bmi-calculator/` (not `/calculators/02-Anorexic bmi-calculator/`)
