ICONS FOLDER
============

Place your favicon and icon files here:

Required files:
- favicon.png (32x32 or 64x64 pixels)
- favicon.ico (for older browser support)
- apple-touch-icon.png (180x180 pixels for iOS)
- og-image.jpg (1200x630 pixels for Open Graph)
- twitter-image.jpg (1200x675 pixels for Twitter Cards)

You can generate favicons using online tools like:
- https://favicon.io/
- https://realfavicongenerator.net/
