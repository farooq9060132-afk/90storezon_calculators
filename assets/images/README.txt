IMAGES FOLDER
=============

Place your website images here:

Recommended images:
- logo.png (transparent background, 200x50 pixels)
- hero-background.jpg (1920x1080 pixels)
- calculator-icons/ (folder for individual calculator icons)
- about-image.jpg (for about page)
- contact-image.jpg (for contact page)

Image optimization tips:
- Use WebP format for better compression
- Compress images before uploading
- Use appropriate dimensions for web
- Add alt text for accessibility

Tools for image optimization:
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (for Mac)
